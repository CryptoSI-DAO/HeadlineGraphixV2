# **App Name**: HeadlineGraphix V2

## Core Features:

- Mock Authentication: Simulate user authentication with a login card that redirects to the dashboard.
- Dashboard Preferences: Allow users to input focus topics and backlink URLs and save them with a toast notification.
- Headline Fetch Simulation: Simulate fetching news headlines with a loading spinner, then display mock news cards.
- Content Pack Studio: Enable users to generate content packs by selecting a brand tone, reference images, and providing a custom angle.
- AI-Powered Draft Generation: Simulate AI generating drafts for blog posts and LinkedIn posts after a loading animation; the tool decides when to incorporate Brand Tone, Reference Images, and User Angle to generate output.
- Reference Image Upload: Allow users to simulate uploading reference images, which are then added to a masonry grid.
- History Tracking: Track generated content packs and display them in a table/list, with a modal for detailed viewing.

## Style Guidelines:

- Primary color: Subtle grayish-blue (#A7C4BC) to evoke a sense of professional design and trust. This avoids design cliches by remaining neutral, allowing content to pop without gaudiness.
- Background color: Very light gray (#F0F4F3) - slightly off-white to reduce harshness.
- Accent color: Muted light-green (#BBD1C1), a neighboring color on the wheel, for interactive elements, conveying balance, sophistication and calm.
- Font: 'Inter' for both headlines and body text. This provides a clean and modern look suitable for a SaaS application focused on content generation.
- Use 'lucide-react' icons with a thin stroke and subtle animation on hover to maintain a clean and professional aesthetic.
- Implement a persistent left sidebar for navigation and a content area with clear separation for inputs and results, using a design studio aesthetic with generous whitespace.
- Use subtle animations for loading states and transitions to enhance the user experience without being distracting.
